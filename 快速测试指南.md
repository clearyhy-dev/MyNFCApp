# 快速测试指南

## 步骤1：准备工作

1. **确保手机已连接电脑**
   ```bash
   adb devices
   ```
   应该能看到你的设备ID

2. **确保手机已开启USB调试模式**
   - 设置 → 关于手机 → 连续点击版本号7次（开启开发者选项）
   - 设置 → 开发者选项 → 开启USB调试

## 步骤2：构建并安装应用

### Windows用户
双击运行 `debug-android.bat`

### 手动执行
```bash
cordova clean android
cordova prepare android
cordova build android --debug
cordova run android --device
```

## 步骤3：调试应用

### 方法一：Chrome DevTools（推荐）

1. 在电脑上打开Chrome浏览器
2. 访问 `chrome://inspect`
3. 找到你的设备，点击 **inspect**
4. 在Console中可以看到调试日志

### 方法二：查看应用内调试面板

1. 应用打开后，点击右下角的 **🐛** 按钮
2. 会显示一个调试日志面板，包含：
   - Cordova就绪状态
   - NFC插件可用性
   - iframe消息传递日志
   - NFC调用日志

## 步骤4：测试远程H5页面

1. **应用会自动加载** `https://49.232.195.86/`

2. **如果远程页面需要调用NFC功能**，需要：
   - 在远程页面中引入NFC桥接脚本（见 `resources/index-remote-example.html` 示例）
   - 使用 `NFCBridge` API调用NFC功能

3. **测试NFC功能**：
   - 初始化插件
   - 查询锁ID和密码
   - 测试开锁/关锁功能

## 常见问题排查

### 问题1：应用无法安装
- 检查设备连接：`adb devices`
- 检查USB调试是否开启
- 尝试重启adb：`adb kill-server && adb start-server`

### 问题2：远程页面无法加载
- 检查手机网络连接
- 在浏览器中访问 `https://49.232.195.86/` 确认可访问
- 查看调试面板的错误信息

### 问题3：NFC功能无法使用
- 查看调试面板，确认：
  - "Cordova已就绪" ✓
  - "NFC插件可用: 是" ✓
- 检查远程页面是否正确使用了NFC桥接API
- 查看Chrome DevTools的Console日志

### 问题4：看不到调试信息
- 点击应用右下角的🐛按钮显示调试面板
- 或者使用Chrome DevTools查看Console

## 调试技巧

1. **查看实时日志**：使用Chrome DevTools的Console
2. **检查网络请求**：在Chrome DevTools的Network标签查看远程页面加载
3. **查看消息传递**：在调试面板中查看所有postMessage消息
4. **NFC调用跟踪**：每个NFC调用都会在调试面板中显示
